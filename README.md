# Projeto: Exemplo de Divs, Menu e CSS

Esta pasta contém uma página de exemplo que atende ao solicitado:

- Aplicar o uso de divs e elementos semânticos para dividir o conteúdo em partes.
- Adicionar um menu de navegação.
- Utilizar CSS para estilização.

Arquivos criados:
- `index.html` - página principal com header, nav, main, aside, seção de cards e footer.
- `styles.css` - estilos para layout, cores e responsividade.

Como usar:
1. Clone o repositório ou baixe os arquivos.
2. Abra `index.html` em um navegador.

Sinta-se à vontade para modificar o conteúdo ou pedir que eu adicione recursos (por exemplo: formulário de contato, imagens, ou integração com JS).
